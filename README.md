# **Digital-Financial-Services-Network-System-Design-and-Implementation**

## **Proposed Network Design and Implementation**
A study by Hamid (2021) highlights that achieving a completely flawless and error-free network is highly unlikely unless the system undergoes thorough evaluation by experts or is tested for vulnerabilities through penetration attempts that simulate potential attackers.

In a hypothetical case study, an organization called DFS consists of four main departments—Human Resources (HR), Information Technology (IT), Customer Service (CS), and Marketing (MK)—all located at its primary headquarters.

To support its operations, I decided to implement an independent network infrastructure. This infrastructure will include a Local Area Network (LAN), a Wide Area Network (WAN), and a separate external server site connected via suitable WAN technologies. A major priority of this design is maintaining secure communication, particularly for data exchanges between the HR department and the external server location. The external site will host essential services such as DNS, web, and email servers, demonstrating the organization’s focus on building a secure and reliable network environment.
